# Kakkoi Nihongo (かっこいい日本語)

**Kakkoi Nihongo** is a free web platform built to help learners master **JLPT N5** — Japan's introductory language proficiency level. It brings together hiragana, katakana, kanji, vocabulary, grammar, quizzes, flashcards, and listening practice in one clean, easy-to-use site.

## ✨ Features

- 🈴 **Hiragana & Katakana** — full character charts with stroke order and pronunciation
- 漢 **Kanji** — all 110 N5 kanji with readings and meanings
- 📚 **Vocabulary & Grammar** — based on *Minna no Nihongo* Lessons 1–25, with Japanese and English explanations
- 📝 **Quizzes & Flashcards** — practice tools to test and reinforce learning
- 🎧 **Listening Practice** — audio-based exercises for real-world comprehension
- 📈 **Progress Tracking** — track your learning journey (with account login)
- 🌗 **Light/Dark Theme** — easy on the eyes, day or night
- 🔍 **Search & Filter** — quickly find kanji, vocab, or grammar points

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Flexbox/Grid, CSS variables), vanilla JavaScript
- **Icons/Fonts:** Lucide/Font Awesome, Noto Sans JP + Poppins/Inter
- **Backend (planned):** Node.js + Express with MySQL/MongoDB
- **Hosting (planned):** Vercel/Netlify/GitHub Pages (frontend), Render/Railway/Fly.io (backend)
